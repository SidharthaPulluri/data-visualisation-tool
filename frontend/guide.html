Guide

How to use the tool step by step.

This page explains the normal workflow in plain English so a first-time user can understand what to do before opening the actual analysis screens.

Step 1

Choose the right start page

  • Use the main page for normal files such as CSV, TSV, DATA, Excel, JSON, and ZIP archives.
  • Use the database page when the dataset came from SQL, MongoDB, or another source system export.
  • If you already uploaded something earlier, you can also resume a saved session or workspace.
Step 2

Upload the dataset

  • Choose the file from your machine.
  • Click Upload and continue.
  • The tool will parse the file, clean obvious issues, and detect the structure before taking you to the preparation page.
Step 3

Read the dataset explanation

  • The tool explains what the dataset appears to contain.
  • It highlights what you can compare, which fields look like counts or rates, and what stands out.
  • This helps you understand the file before you start charting.
Step 4

Prepare the data view

  • Review the schema, preview table, and data quality hints.
  • Optionally filter rows, keep only some columns, group values, or create a derived column.
  • If you make a mistake, use the transformation history and undo controls.
Step 5

Create visuals

  • Move to the visualize page after the prepared view looks right.
  • Use the recommended chart setup first if you are unsure.
  • You can save multiple charts, switch between them, view them in a grid, and add notes.
Step 6

Use cross-filtering and explanations

  • Click saved-chart filter chips to narrow the preview table from the chart itself.
  • Read the plain-English explanation under the chart to understand what it shows and how to read it.
  • Use grid view when you want to compare several visuals together.
Step 7

Export the results

  • Download the cleaned dataset as CSV or Excel.
  • Download one chart or export a dashboard PDF.
  • Download a report if you want a text summary of the analysis.
Tips

How to get better results

  • Use top-N filtering when category labels are too many.
  • Do not sum percentages or rates unless you really mean to.
  • Prefer bar or line charts first if you are not sure what to pick.
  • Save workspaces when you want to come back later without repeating the setup.
Chart help

Learn what each graph is good for

  • Open the separate chart guide when you want help choosing between bar, line, heatmap, bubble, hexbin, feature graph, and the other chart types.
  • It explains what each chart is used for, how to read it, and when it is a weak choice.
  • This is helpful before the visualize step if you are unsure which graph matches your question.
