Section 1 - The Problem

Heading
Most file-based analysis starts with confusion.

Body
When someone gets a spreadsheet or CSV, the hardest part is not creating a chart. It is understanding the dataset itself.

People usually do not know:
- which column is the main category
- which fields are counts versus rates
- whether a pie chart or bar chart makes sense
- what metric should be used first
- whether the summary they are seeing is misleading

This tool helps users get oriented before they make bad charts or shaky conclusions.


Section 2 - The Promise

Heading
From upload to understanding in minutes.

Body
The tool reads structured datasets with deterministic rules and guides users through:
- dataset explanation
- data quality checks
- chart recommendations
- plain-English insights
- dashboard-style exploration
- exportable outputs

Everything is designed to help users move from raw file to useful understanding quickly and safely.
